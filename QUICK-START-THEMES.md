# 🚀 Quick Start: Dynamic Themes

## ⚡ Get Started in 3 Steps

### 1️⃣ Install Dependencies (if needed)
```bash
cd food-of-indians
npm install
```

### 2️⃣ Run Development Server
```bash
npm run dev
```

### 3️⃣ Open Browser
```
http://localhost:5173
```

**That's it!** The dynamic theme system is already integrated and working.

---

## 🎮 How to Use

### Automatic Mode (Default)
- Just browse the site normally
- Theme changes automatically based on time
- No action required from you

### Manual Control
1. Look for the **palette icon** (🎨) in bottom-right corner
2. Click it to open theme options
3. Choose any theme you like
4. Click "Auto" to re-enable automatic switching

## 🐛 Troubleshooting

### Theme Not Changing?
- Check if you're in manual mode (theme switcher)
- Click "Auto" to re-enable automatic switching
- Refresh the page

### Colors Look Wrong?
- Clear browser cache
- Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
- Check browser console for errors

### Theme Switcher Not Visible?
- Scroll down a bit (it's fixed position)
- Check bottom-right corner
- Try zooming out if needed
