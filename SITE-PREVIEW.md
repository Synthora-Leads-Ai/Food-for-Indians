## 🎯 Signature Interactive Elements

### 1. 3D Lotus (Hero)
- Three.js torus knot geometry
- Saffron/gold colors
- Auto-rotates slowly
- Floats up and down
- Glowing effect

### 2. Floating Action Buttons (All Pages)
- Fixed bottom-right
- Three buttons stacked:
  - WhatsApp (green)
  - Call (saffron)
  - Scroll to Top (gold, appears after scrolling)
- Hover: scale up
- Glow effect

### 3. Navbar
- Fixed top
- Transparent initially
- Becomes solid cream with shadow on scroll
- Logo: "Food of Indians" + "by Jain Delight"
- Desktop: horizontal links
- Mobile: hamburger menu
- "Call Now" button (desktop only)

### 4. Footer
- Dark maroon/brown gradient
- 4 columns:
  - Brand info
  - Quick links
  - Contact info
  - Opening hours
- Bottom bar: Copyright + "Made with 🙏 in Jabalpur"

## 📱 Responsive Behavior

**Mobile (< 768px):**
- Single column layouts
- Hamburger menu
- Stacked buttons
- Smaller text sizes
- Touch-friendly spacing

**Tablet (768px - 1024px):**
- 2-column grids
- Adjusted spacing
- Readable text sizes

**Desktop (> 1024px):**
- Full multi-column layouts
- Horizontal navigation
- Larger images
- Optimal reading width (max-w-7xl)

## ✨ Micro-interactions

1. **Button Hover:** Background darkens, slight scale
2. **Card Hover:** Lifts up (translateY), shadow increases
3. **Image Hover:** Scales to 110%, smooth transition
4. **Link Hover:** Underline expands from left
5. **Form Focus:** Border changes to saffron
6. **Scroll:** Elements fade in from bottom
